# Bun SQLite ElysiaJS API Example Project
This project was created using `bun init` in bun v1.1.16. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.

## How to 
Install dependencies:
```bash
bun install
```

Test:
```bash
bun test
```

Run:
```bash
bun start
```
